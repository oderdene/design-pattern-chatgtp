### üîπ **8. How is the command pattern implemented in a programming language you know?**

–ñ–∏—à—ç—ç –±–æ–ª–≥–æ–∂ Python —Ö—ç–ª –¥—ç—ç—Ä —Ö—ç—Ä—ç–≥–∂“Ø“Ø–ª–∂ “Ø–∑—å–µ:

```python
# Command Interface
class Command:
    def execute(self):
        pass

# Receiver
class TV:
    def turn_on(self):
        print("–¢–í –∞—Å–∞–∞–≤.")
    def turn_off(self):
        print("–¢–í —É–Ω—Ç—Ä–∞–∞–≤.")

# ConcreteCommand
class TurnOnCommand(Command):
    def __init__(self, tv):
        self.tv = tv
    def execute(self):
        self.tv.turn_on()

class TurnOffCommand(Command):
    def __init__(self, tv):
        self.tv = tv
    def execute(self):
        self.tv.turn_off()

# Invoker
class RemoteControl:
    def __init__(self):
        self.command = None
    def set_command(self, command):
        self.command = command
    def press_button(self):
        self.command.execute()

# Client
tv = TV()
on_command = TurnOnCommand(tv)
off_command = TurnOffCommand(tv)

remote = RemoteControl()

remote.set_command(on_command)
remote.press_button()  # -> –¢–í –∞—Å–∞–∞–≤.

remote.set_command(off_command)
remote.press_button()  # -> –¢–í —É–Ω—Ç—Ä–∞–∞–≤.
```

### üîπ **9. Are there alternative ways to implement the Command Pattern?**


#### ‚úÖ **Functional Command Queue**  
Command-—É—É–¥—ã–≥ –∂–∞–≥—Å–∞–∞–ª—Ç –±–æ–ª–≥–æ–Ω —Ö–∞–¥–≥–∞–ª–∞–∞–¥ –¥–∞—Ä–∞–∞ –Ω—å batch-–∞–∞—Ä –≥“Ø–π—Ü—ç—Ç–≥—ç—Ö.
